# Welcome to Repack-your Environment Sustainable Dashboard 

## Project info
This is a Dashboard to help promote environmentally sustainable practices in the packaging and logistics industry. The Goal of this project was to provide alternatives that would reduce the carbon footprint and overall general harm that traditional practices cause. This is my small contribution towards a green revolution and I hope for a greener tomorrow. 

##  Steps to run/visualize the dashboard:
```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?
This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
